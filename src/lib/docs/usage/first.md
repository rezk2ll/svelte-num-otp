# Installing

```
npm install @kferjani/svelte-num-otp
```

or

```
yarn add @kferjani/svelte-num-otp
```

<br/>
<br/>

# Usage

## Simplest form

Getting started is as easy as importing the `SvelteOtp` component and just using it.

<br/>

```js
<script>
    import SvelteOtp from ‘@kferjani/svelte-num-otp’;
</script>

<SvelteOtp />
```
